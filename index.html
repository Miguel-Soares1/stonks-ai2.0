Quick Start 
# 1. Instalar
pip install stonks-ai

# 2. Inicializar
stonks init

# 3. Consultar cotação
stonks quote PETR4

# 4. Análise com IA
stonks analyze PETR4

# 5. Comparar ações
stonks compare PETR4 VALE3 ITUB4

# 6. Web app local
streamlit run webapp/app.py
